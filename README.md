# ITI PD46 – JavaScript Final Task

A front-end web application developed as a final task for the ITI PD46 program, demonstrating core JavaScript concepts along with clean UI design and structured user flow.

## Project Overview
This project is a simple yet complete web application that includes authentication pages and interactive functionality implemented using vanilla JavaScript.  
The focus of the project is on functionality, user experience, and proper separation of structure, style, and logic.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure
- `index.html` – Main page  
- `login.html` – Login screen  
- `signup.html` – Registration screen  
- `details.html` – Details page  
- `style.css` – Application styling  
- `script.js` – Application logic
